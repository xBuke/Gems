SpaceMono section headers
Section headers ("ACCOUNT", "PRIVACY"…) changed to SpaceMono. Current style is plain sans-serif — the only place in the app where section labels aren't monospace. One font change, consistent system.
2
Premium status card
Settings currently has no subscription awareness. "Explorer Lv. 2 · Premium Active" banner at top surfaces tier and level — or shows "Upgrade" for free users. Reduces confusion about what's included.
3
Dark mode toggle → connected live
The toggle above is wired — tap it to switch all phones to light/dark mode simultaneously. Danger Zone gets a subtle red-tinted background to visually distinguish it from normal settings rows.

Users / Gems / Active / Reports counters in SpaceMono monospace — consistent with how all numeric data is treated across the app. The red Reports card with border creates instant priority visibility.

Reports surfaced as a priority list with inline Review/Done CTAs. Currently the admin screen is a scrollable list — the most urgent items should be front and center, not buried.
