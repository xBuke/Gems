1
SpaceMono coordinates in card
Meta line shows @buke · 43.0512° N, 17.4291° E · 4.2 km. Currently only username + distance. Coordinates are the app's signature detail — they belong on every gem surface.
2
Deck counter
"12 gems remaining" in SpaceMono above buttons. Users have no sense of deck depth — they don't know if they're about to run out. Tap ♥ or ✕ to see the stamps animate.
3
Three-button bar with view detail
Added a smaller 💎 centre button to open the gem's full detail page. Currently the only way to see more about a card is to save it first — this adds a low-commitment "peek" action.

Coral avatar ring
Header avatar gets a coral ring: box-shadow: 0 0 0 2px bg, 0 0 0 3.5px coral. Matches the Profile avatar ring — consistent social identity pattern across the app.

📍 Gem share button in input
A pin button next to the input opens a gem picker to share directly in chat. Coordinates shared in SpaceMono look great as bubbles — see the 43.0512° bubble above.
3
SpaceMono date separator + coord bubbles
"TODAY" in SpaceMono 10px monospace vs current system 12px sans. When a user sends coordinates in a message, rendering them in SpaceMono teal inside the teal bubble looks intentional, not accidental.
