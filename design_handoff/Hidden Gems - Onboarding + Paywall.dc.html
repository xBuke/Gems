Compass replaces checkmark
Generic ✓ icon replaced with the app's own compass motif (teal ring, coral N-needle). Consistent with the auth screen logo and the app icon. Last screen should reinforce the brand, not use a system symbol.

"Welcome, Explorer" + coords

Button with glow shadow
