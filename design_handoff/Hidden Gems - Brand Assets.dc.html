1
expo-splash-screen setup
resizeMode: "contain", backgroundColor: "#0B1A1C". The static PNG is just the compass centred. Hold the splash until fonts + auth state are resolved, then animate out.
2
Coordinates typing animation
After the splash PNG is dismissed, a React Native screen fades in with TypeWriter-style animation on the coords. Use user's last known location — or the default 43° N / 17° E if unavailable. Tap ↺ above to preview.
3
No wordmark on splash
The app name is already on the home screen icon label. A wordmark on the splash is redundant — the compass alone is enough identity. Coordinates as the only text makes it feel like a field instrument powering up.

1
Teal = skill / coral = identity
Teal badges (Pioneer, Navigator, Pathfinder, Trailblazer, Secret Find) = exploration activity. Coral badges (Local Legend, Founding, Connector) = social identity and status. Consistent with the app's teal/coral duality throughout.
2
One locked-state style for all
All 8 badges share a single AchievementBadge component. locked={true} applies grayscale(1) filter + opacity 0.4. No duplicate icon assets needed.
3
Unlock: hapticSuccess + flash
Badge unlock triggers hapticSuccess(), a 0.5s full-screen teal flash (opacity 0→0.15→0), then a bottom sheet showing the badge. No pop-up mid-session — queue the sheet to show at session end.

Fallback (no photo)
When a gem has no photo, the left column renders as a dark gradient with the compass watermark only. Same layout, different left panel fill.
1
Generated server-side via Satori
Use @vercel/og (Satori under the hood) at /api/og?gemId=.... Fetches gem data + photo URL, renders to PNG. Space Grotesk + Space Mono both available as fetch() font inputs.
2
"Open in App" deep link
The CTA links to hiddengems://gem/[id] (opens app) with https://hiddengems.app/gem/[id] as fallback (opens web preview with App Store CTA). Universal Links for iOS, App Links for Android.
3
Also used for push notification icon
Android requires a monochrome notification-icon.png (white on transparent). Export the compass SVG as a single-colour white SVG → PNG at 96×96px and add to app.json → expo.android.icon.
